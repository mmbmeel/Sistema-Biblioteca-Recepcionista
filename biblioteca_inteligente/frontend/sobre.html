Biblioteca Gestão Inteligente
Um sistema web completo para o gerenciamento de bibliotecas acadêmicas — com controle de acervo, usuários, empréstimos, reservas e relatórios, tudo em uma interface moderna e intuitiva.
v1.0.0 Trabalho Acadêmico Programação II Fullstack
🔧 Tecnologias Utilizadas
🟩
Node.js
Runtime JavaScript para o servidor — rápido, assíncrono e escalável.
⚡
Express.js
Framework minimalista para criação da API REST que alimenta o frontend.
🐬
MySQL
Banco de dados relacional que armazena livros, usuários, empréstimos e reservas.
🌐
HTML5
Estrutura semântica de todas as páginas da aplicação, com formulários acessíveis.
🎨
CSS3 / Vanilla CSS
Design system próprio com variáveis CSS, glassmorphism, gradientes e animações.
✨
JavaScript ES6+
Toda a lógica de frontend — CRUD, máscaras, validações e comunicação com a API.
📦
LocalStorage API
Fallback automático quando o servidor está offline, mantendo a usabilidade.
🔗
REST API
Comunicação frontend ↔ backend via endpoints HTTP padronizados com JSON.
🏗️ Arquitetura do Sistema
🌐   Frontend  —  HTML + CSS + JavaScript Vanilla páginas estáticas servidas pelo Express
⬆ ⬇  HTTP / REST JSON
⚡   Backend  —  Node.js + Express.js API REST · porta 3000
⬆ ⬇  mysql2 driver
🐬   Banco de Dados  —  MySQL livros · usuários · empréstimos · reservas
⚙️ Funcionalidades do Sistema
📊
Dashboard
Visão geral em tempo real com KPIs de livros, usuários, empréstimos e reservas. Destaque dos livros mais recentes do acervo.
📚
Catálogo de Livros
Navegação pelo acervo com filtros por categoria, idioma e status. Visualização em cards com capa, sinopse e informações detalhadas.
➕
Cadastro de Livros
Cadastre, edite e exclua livros com suporte a upload de capa (base64), ISBN, sinopse, idioma, editora e controle de quantidade disponível.
👤
Cadastro de Usuários
Gerenciamento de leitores com máscara automática de CPF (###.###.###-##), validação dos dígitos verificadores e formatação de telefone celular.
🔄
Empréstimos
Registre, edite e gerencie empréstimos com seletor pesquisável de livros e leitores. Detecção automática de atrasos e devolução com um clique.
🔖
Reservas
Reserve livros para leitores. Ao concluir uma reserva, o sistema cria automaticamente o empréstimo correspondente — sem retrabalho.
📊
Relatórios
Gráficos de barras com distribuição de livros por categoria e idioma, usuários por tipo e status dos empréstimos. Dados atualizados em tempo real.
📡
Modo Offline
Fallback automático para LocalStorage quando o servidor não está disponível. O sistema continua funcional com dados de demonstração.
🚀 Como Usar a Plataforma
1
Inicie o servidor backend
Instale as dependências com npm install e inicie com npm start ou node server.js. O servidor ficará disponível em http://localhost:3000.
2
Configure o banco de dados MySQL
Crie o banco e execute o script de criação das tabelas disponível no repositório. Configure as credenciais no arquivo .env (DB_HOST, DB_USER, DB_PASS, DB_NAME).
3
Cadastre o acervo de livros
Acesse Cadastro de Livros no menu lateral. Clique em "Novo Livro", preencha os dados e opcionalmente faça upload da capa arrastando uma imagem para o campo indicado. Salve para adicionar ao acervo.
4
Cadastre os leitores
Em Cadastro de Usuários, registre os leitores com nome, e-mail, CPF e telefone. O sistema valida automaticamente o CPF e aplica a máscara no telefone celular ((##) # ####-####).
5
Registre empréstimos e reservas
Use Empréstimos para registrar a saída de livros. Pesquise o livro e o leitor pelo nome no seletor. Para reservas antecipadas, use a página Reservas — ao concluí-la, o empréstimo é criado automaticamente.
6
Registre devoluções
Na tabela de empréstimos, clique no botão "↩ Devolver" para registrar a devolução do livro. A data de devolução real é preenchida automaticamente com a data atual.
7
Acompanhe pelos Relatórios
Na página de Relatórios, visualize gráficos com o estado atual da biblioteca — distribuição por categorias, idiomas, tipos de usuários e status dos empréstimos.

Trabalho Acadêmico — Programação II

Este sistema foi desenvolvido como projeto acadêmico, demonstrando a integração completa de um backend RESTful em Node.js com Express e MySQL e um frontend responsivo desenvolvido integralmente em HTML, CSS e JavaScript Vanilla, sem o uso de frameworks pesados. O design segue princípios modernos de UI/UX com glassmorphism e animações suaves.
